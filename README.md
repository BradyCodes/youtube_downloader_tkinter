# Installation: Virtual Enviorment

### Setting Up Your Virtual Enviorment

    cd Desktop/
    python3 -m venv ytd
    cd ytd
    source bin/activate

### Installing With Git

    cd ../
    git clone git@github.com:BradyCodes/youtube_downloader_tkinter.git

### Installing The Packages

    pip install -r requirements.txt
